# Leo X.

Personal blog powered by [Hugo](https://gohugo.io) and [Minima](https://github.com/mivinci/hugo-theme-minima).

## Setup

```bash
# Install Hugo (extended)
brew install hugo

# Preview locally
hugo server

# Build for production
hugo --gc --minify
```

## Deploy

Push to `main` branch, GitHub Actions automatically builds and deploys to GitHub Pages.
